# **App Name**: SME DataBrain

## Core Features:

- Data Transformation: Convert uploaded documents and images into structured data suitable for analysis, using AI to interpret handwritten notes and unstructured formats. Act as a tool deciding what is necessary from each document.
- Business Health Scoring: Generate an instant business health score based on uploaded data, highlighting strengths and weaknesses.
- Performance Dashboard: Visualize key performance indicators (KPIs) such as revenue trends, top products, expense breakdown, and inventory flow in an interactive dashboard with animated charts and graphs.
- AI Assistant: Enable users to ask natural language questions about their business performance and receive instant insights from the AI assistant.
- Automated Reports: Generate weekly alerts, monthly summaries, branch comparisons, inventory anomaly detection, and automated expense categorization to proactively monitor business operations.
- Onboarding Experience: Guide new users through a clean, simple, and futuristic onboarding process, including business registration and record upload.
- Video Persona Integration: Integrate a video persona (Tavus Integration) that greets users, explains insights, and provides monthly summaries like a friendly operations analyst.

## Style Guidelines:

- Primary color: Magenta (#FF00FF) for a vibrant and futuristic feel, nodding to the soft neon highlights specified in the user prompt.
- Background color: Dark grey (#222222) for a minimalistic dark-mode-ready interface.
- Accent color: Aqua (#00FFFF) to highlight interactive elements and CTAs, contrasting with the magenta.
- Body and headline font: 'Space Grotesk', modified with a subtle pixelation effect similar to the hero image, creating a next-gen feel. For supporting text, use a cleaner font like 'Inter' for readability.
- Use simple, glowing line icons for the sidebar and throughout the dashboard. The glow should be subtle and not overly exaggerated.
- Implement a clean, card-based structure with balanced spacing and modern grid alignment, optimized for mobile-first responsiveness.
- Incorporate smooth transitions, animated data uploads, and subtle hovering glows to enhance user interaction and create a futuristic feel. Hero section animation inspired by the provided image, featuring horizontally sliding cards with SME-themed imagery.
